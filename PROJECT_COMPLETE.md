## 🚀 How to Get Started

### Step 1: Start Development Server (1 minute)
```bash
cd /vercel/share/v0-project
pnpm install
pnpm dev
```

### Step 2: View Your Portfolio (Instant)
Open [http://localhost:3000](http://localhost:3000) in your browser

### Step 3: Customize Content (5-10 minutes)
Edit `/lib/portfolio-data.ts` with your information:
- Your name, title, email
- Social media links
- Projects, skills, experience
- Testimonials, blog posts

### Step 4: Deploy (5-15 minutes)
Follow the simple steps in `DEPLOYMENT.md` to deploy to Vercel

## 🔧 Customization

### Edit Your Information
```typescript
// /lib/portfolio-data.ts
export const portfolioOwner = {
  name: 'Your Name',
  title: 'Your Job Title',
  email: 'your@email.com',
  // ... more fields
};
```

### Change Colors
Edit `/app/globals.css` to update:
- Primary color (cyan)
- Secondary color (blue)
- Background color
- Text colors

### Add Your Images
1. Add project images to `/public/projects/`
2. Add testimonial avatars to `/public/avatars/`
3. Update image paths in portfolio data

### Update Social Links
Edit social media links in `portfolioOwner` object
