## 💡 Customization Quick Tips

### Change Your Name
Edit `/lib/portfolio-data.ts` → `portfolioOwner.name`

### Change Primary Color
Edit `/app/globals.css` → Replace `cyan` with your color

### Add a Project
Add to `projectsData` array in `/lib/portfolio-data.ts`

### Update Skills
Modify `skillsData` array in `/lib/portfolio-data.ts`

### Update Experience
Modify `experienceData` array in `/lib/portfolio-data.ts`

### Add Testimonials
Add to `testimonialsData` array in `/lib/portfolio-data.ts`
